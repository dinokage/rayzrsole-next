# RayzrSole — Landing Page

A coming soon landing page for [RayzrSole](https://rayzrsole.com), built with
[Next.js](https://nextjs.org) and Tailwind CSS v4.

## Getting started

1. Install the dependencies. We recommend using pnpm. If you want to use `npm`,
   just replace `pnpm` with `npm`.

```bash
pnpm install
```

2. Then, start the development server:

```bash
pnpm run dev
```

3. Visit [http://localhost:3000](http://localhost:3000) in your browser to view
   the template.

## Notes

This project uses
[`next/font`](https://nextjs.org/docs/basic-features/font-optimization) to
automatically optimize and load Inter, Playfair Display, and Poppins fonts.
